# FS25 Slurry Extension Loading Point

Support and issue tracker for the Farming Simulator 25 mod **Slurry Extension Loading Point** by CrankyAnt.

This repository is intentionally documentation-only. It is used for bug reports, compatibility notes, and release notes. The mod package and game assets are not hosted here.

## Download

The official download is available through GIANTS ModHub once published.

## What This Mod Does

Adds a placeable loading point for slurry extension storages.
It is meant for farms where liquid manure, digestate, or other slurry tank fill types are already stored in nearby extensions, but no separate loading point is available for vehicles or AI helpers.
Place it near your slurry storage and use it as a simple pickup point without adding extra storage capacity.
The loading point supports the slurry tank fill type category, so compatible fill types added by maps or other mods can be handled through the same placeable.

## Compatibility

- Farming Simulator 25
- Multiplayer supported
- PC, Mac, PlayStation, and Xbox supported

## Reporting Issues

Found a bug or compatibility issue? Please open a GitHub issue:

https://github.com/CrankyAnt/FS25_SlurryExtensionLoadingPoint/issues/new/choose

Please open an issue and include:

- Farming Simulator 25 game version
- Platform: PC, Mac, PlayStation, or Xbox
- Map name
- Other relevant mods active in the savegame
- A short description of what happened and what you expected
- The game log, if the issue involves errors or missing functionality
